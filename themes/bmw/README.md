## 特性一览

- 写文章 · 随心而动
  - 支持**目录渲染/关闭**
  - 支持**多平台分享**
  - 支持**用户打赏**
  - 支持图片**幻灯片播放**
  - 支持**数学公式渲染**
  - 支持您为不同文章自定义的**音乐播放器**

- 多页面 · 一键启动
  - 支持**二级导航栏**
  - 自带**友链页面**
  - 自带**关于页面**
  - 自带**标签云页面**
  - 自带**分类云页面**
  - 支持**自定义页面**

- 社交 · 不止你我
  - 自带**评论系统**
  - 自带**文章统计插件**

- 版权 · 保护原创
  - 支持文章**版权声明和相关信息自定义**
  - 支持自定义**页脚个人信息**

- 自定义 · 个性彰显
  - 支持**自定义样式文件**
  - 支持**自定义脚本文件**

- 其它 · SEO | 极速 | ...
